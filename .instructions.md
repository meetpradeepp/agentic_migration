- **Requirements Gathering**: [.github/skills/requirements-gathering/SKILL.md](.github/skills/requirements-gathering/SKILL.md)

- **Complexity Assessment**: [.github/skills/complexity-assessment/SKILL.md](.github/skills/complexity-assessment/SKILL.md)

  - Three-tier complexity system (simple/standard/complex)

## Adding New Instructions

When adding new instruction files:

1. Create them in `.github/instructions/` using the naming pattern: `<topic>.instructions.md`
2. Update this file to reference the new instructions
3. Follow the structure and style of existing instruction files
4. Be specific and actionable in your guidelines

## File Discovery

**Note**: Root-level instruction files (like this one) are discovered most reliably by AI tools. This file serves as a pointer to ensure all instruction files in `.github/` are properly referenced and discoverable.
